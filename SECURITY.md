# Security Policy

Security is always the most important thing, although this project is so tiny.

We believe our code should let people happy, so we are happy to improve our code's security.

## Reporting a Vulnerability

If you believe you have found a security issue: 

- **DO NOT** just open an issue directly.
- Instead, email to `pctgroup@163.com`.

We will handle it as soon as possible.

## Tips

Because we are just a small group of students, not a community or professional developers, so maybe we could not review the issue so quickly.

Thanks for your patience.

P.C.T.G.
